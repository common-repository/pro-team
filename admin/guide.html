Introduction

Hello. this plugin composes of three sections. Admin Panel (this one), Members manager and a shortcode manager. To access admin panel and Member manager, you must click the proper option at left, under admin panel. To use the shortcode manager, you must create or modify a page and above wp editor you will see a button named "ProTeam"

Setting Defaults

Here, in admin panel, you will set some values. This values will define the default ones, so each time that you add a new member, the starter value of each value it will be the same as here. Also, when you insert the shortcode, you could choose between use the values defined here, or use the values defined under each member.

Add Member

Navigate to ProTeam, then press add new. Title:This field is used to display the member name Editor:Used to display the member description, under full bio. Editor:Used to display the member description, under full bio. Features:Here you can add members skill, or whatever you want. You can choose background color and text color, and rearrange them pressing arrows. You can add as many as you want. Social Options:Here you can add social icons with proper link. You can choose background color and icon color, and rearrange them pressing arrows. You can add as many as you want. Design Options: Stylize your member. If you define different setting for each member, you must select the proper option when you insert the shortcode. Check the next section.

Insert Shorcode

Navigate to pages->Add new (or modify an existing one). Press the ProTeam Button (Above editor). Style: If you choose custom, it will use the style defined under each member options, otherwise, it will use the same style to all members displayed with this shortcode. Design: If you choose default, it will use the style defined under admin panel, custom will use the styles defined under member page. Column: Members column. Members per page: Number of members that will be displayed in this page. Orderby and order: This define the order. Ex: If you choose date and ASC, it will display members from oldest to newest Spaces between members:Pretty obvious. Display Pagination:Ii will be displayed only if applies.Ex: If you added 16 members, and select "members per page = 12", pagination will appear. Custom Class (Optional): The custom class it will be added to each member markup, so it will affect member list and member bio.
